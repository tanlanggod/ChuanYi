# ChuanYi Admin

Vue3 + Vite + Element Plus 管理后台。

## 启动

```bash
npm install
npm run dev
```

## 打包

```bash
npm run build
```

## 已实现页面

1. `/login` 登录页
2. `/` 数据看板
3. `/orders` 订单管理（查询/详情/取消/发货/完成）
4. `/goods` 好物管理（SPU/SKU 编辑、上下架）
5. `/components` 素材库管理（分类管理 + SKU 管理）
6. `/content` 内容管理（Banner/设计广场、图片上传）
